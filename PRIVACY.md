# Privacy Policy - Hide My Name

**Last updated:** March 20, 2026

## Overview

Hide My Name is a Chrome browser extension that replaces user-specified words and phrases on web pages with a chosen display name. It is designed for live streamers who want to protect their personal information while sharing their screen.

## Data Collection

Hide My Name does **not** collect, transmit, or share any user data. Specifically:

- **No personal information** is collected.
- **No browsing history** is tracked or recorded.
- **No website content** is sent to any server.
- **No analytics or telemetry** data is gathered.
- **No cookies** are set by the extension.
- **No network requests** are made by the extension.

## Data Storage

The extension stores the following data locally using Chrome's built-in `chrome.storage.sync` API:

- A list of words/phrases the user wants to hide
- Replacement text for each word/phrase
- A global display name
- An enabled/disabled toggle

This data is stored entirely within Chrome's local extension storage and is synced across the user's own Chrome profile if they are signed in. It is never sent to any external server or third party.

## Third Parties

Hide My Name does not share any data with third parties. There are no external services, APIs, or servers involved.

If this privacy policy changes, the updated version will be posted to this repository.
